Slider uk-slider

  • uk-slider - Create a list of items to use as a responsive carousel slider
  • add the data-uk-slider attribute to a container element around a .uk-slider-container element
  • data-uk-slider="{center:true}" - to center the elements
  • uk-grid - add the .uk-grid class to the slider container for spacing between the elements

Slideset data-uk-slideset

  • data-uk-slideset - Create sets and groups of items, allowing to loop through the sets.
