Slider uk-slider

  • uk-slider - Create a list of items to use as a responsive carousel slider
  • add the data-uk-slider attribute to a container element around a .uk-slider-container element
  • data-uk-slider="{center:true}" - to center the elements
  • uk-grid - add the .uk-grid class to the slider container for spacing between the elements

Slideset data-uk-slideset

  • data-uk-slideset - Create sets and groups of items, allowing to loop through the sets.
  • animation - add the "animation" option to the "data-uk-slideset". For e.g "data-uk-slideset="{default: 4, animation: 'slide-horizontal'}""

Slideshow uk-slideshow

  • uk-slideshow - Create a responsive image or video slideshow with stunning transition effects, fullscreen mode and overlays.
  • add the data-uk-slideshow attribute in order to load the necessary JavaScript

Overlays uk-overlay-panel | uk-overlay-background | uk-overlay-fade

  • uk-overlay-panel - Enhance slideshows with different overlays.
  • uk-overlay-background - to show a background
  • uk-overlay-fade - to show a transition
